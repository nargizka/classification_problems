# Wine Quality Classification 
- `./scripts/test_rf.py`: Uses Random Forest Classifier to predict wine quality based on chemical features.
- `./scripts/test_rf_smote.py`: Extends the RF Classifier by applying SMOTE to handle class imbalance and improve detection of the minority class (high quality wines).
- dataset source: https://archive.ics.uci.edu/dataset/186/wine+quality
